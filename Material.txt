MVC Architecture

               Applications
                /    |    \
              /      |     \
             M       V      C


Model

--Think of the model as data
--JSON, database data etc


View
--Used to display content & data to a user in a browser
--HTML
--Use expressions to insert data into View
<html tag> {{data}}</html tag>


Controllers
--Control the functionality of our View
--Performs the interaction between our model and our View
--Difference controllers for different areas in our applications.



User clicks a  show User button in a View
The usercontroller recognises the button click event and performs a function
The function communicates with the 'users' model and retrieves all the use data
The controller passes accessibility to this data to the view, which then display
it to the end user via expressions.
